# 📝 Task Manager API

Junior darajadagi loyiha uchun mukammal Backend tizimi. Ushbu loyiha yordamida foydalanuvchilar o'z loyihalarini boshqarishlari va ularga vazifalar (rasm yuklash imkoniyati bilan) biriktirishlari mumkin.

## 🚀 Demo
Loyihani real vaqtda sinab ko'ring:
- **Swagger Documentation:** [https://task-manager-api-jdbr.onrender.com/api-docs/](https://task-manager-api-jdbr.onrender.com/api-docs/)

---

## ✨ Imkoniyatlar
- **Autentifikatsiya:** JWT (JSON Web Token) orqali himoyalangan kirish va ro'yxatdan o'tish.
- **Project Management:** Loyihalarni yaratish va ko'rish.
- **Task Management:** Vazifalarni boshqarish va har bir vazifaga rasm yuklash.
- **Security:** Parollarni `bcrypt` yordamida shifrlash.
- **Documentation:** Interaktiv Swagger UI.

---

## 🛠 Texnologiyalar
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas (Mongoose)
- **File Upload:** Multer
- **Validation:** Express-validator
- **Deployment:** Render
